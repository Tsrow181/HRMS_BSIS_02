# Philippine Leave Types - Quick Reference Guide

**For HR Staff & Managers | Valid as of February 18, 2026**

---

## Leave Types at a Glance

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ 1. VACATION LEAVE (RA 10911)                                                │
├─────────────────────────────────────────────────────────────────────────────┤
│ Default Days:        15 days per year                                        │
│ Carry Forward:       Yes (Max 5 days to next year)                          │
│ Paid/Unpaid:         PAID ✓                                                 │
│ No. Required/Year:   At least 10 days must be taken                         │
│ Medical Cert:        Not required                                           │
│ For:                 Rest, recreation, family time                          │
│ Approval Level:      Manager                                                │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 2. SICK LEAVE (RA 10911)                                                    │
├─────────────────────────────────────────────────────────────────────────────┤
│ Default Days:        15 days per year                                        │
│ Carry Forward:       Yes (Max 5 days to next year)                          │
│ Paid/Unpaid:         PAID ✓ (with medical cert for >2 days)                 │
│ Medical Cert:        Required if >2 consecutive days                         │
│ For:                 Personal illness / Family member care                   │
│ Approval Level:      Manager (HR for medical cert)                          │
│ SENSITIVE DATA:      ⚠️ Health information (RA 10173)                        │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 3. MATERNITY LEAVE (RA 11210)                                               │
├─────────────────────────────────────────────────────────────────────────────┤
│ Default Days:        120 days (can be split)                                │
│ Carry Forward:       No                                                      │
│ Paid/Unpaid:         PAID ✓ (100% of salary)                                │
│ Eligibility:         All female employees                                    │
│ Medical Cert:        Birth certificate (2 weeks before due date)            │
│ For:                 Childbirth care and recovery                           │
│ Approval Level:      HR Director + Medical Review                          │
│ SENSITIVE DATA:      ⚠️⚠️ Health information (RA 10173)                      │
│ Timeline:            Valid up to 5 years from birth date                    │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 4. PATERNITY LEAVE (RA 11165)                                               │
├─────────────────────────────────────────────────────────────────────────────┤
│ Default Days:        7 days (14 days for solo parents)                      │
│ Carry Forward:       No                                                      │
│ Paid/Unpaid:         PAID ✓ (100% of salary)                                │
│ Eligibility:         All male employees                                      │
│ Medical Cert:        Birth/Adoption certificate                             │
│ For:                 Childbirth support / Adoption                          │
│ Approval Level:      HR Director                                            │
│ SENSITIVE DATA:      ⚠️ Family status information (RA 10173)                  │
│ Deadline:            Must claim within 60 days of birth                     │
│ NOTE:                Solo parents get additional 5 days via Solo Parent     │
│                      Leave benefit (total = 14 days)                        │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 5. SOLO PARENT LEAVE (RA 9403)                                              │
├─────────────────────────────────────────────────────────────────────────────┤
│ Default Days:        5 days per year (additional)                           │
│ Carry Forward:       No                                                      │
│ Paid/Unpaid:         PAID ✓                                                 │
│ Eligibility:         Registered solo parents (male & female)                │
│ Medical Cert:        DSWD Solo Parent Certificate or LGU registration       │
│ For:                 Childcare, medical needs, family emergencies           │
│ Approval Level:      Manager (HR verifies certification)                    │
│ SENSITIVE DATA:      ⚠️ Family status information (RA 10173)                  │
│ Usage:               Flexible - can be used any time                        │
│ COMBINED with:       Maternity/Paternity (solo parents get all benefits)    │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 6. MENSTRUAL DISORDER LEAVE (RA 11058)                                      │
├─────────────────────────────────────────────────────────────────────────────┤
│ Default Days:        Up to 3 days per year                                  │
│ Carry Forward:       No (annual entitlement)                                │
│ Paid/Unpaid:         PAID ✓                                                 │
│ Eligibility:         Female employees with medically diagnosed menstrual    │
│                      disorders                                               │
│ Medical Cert:        Required - License physician diagnosis (one-time)      │
│ For:                 Severe menstrual symptoms/dysmenorrhea                 │
│ Approval Level:      Manager (HR for medical cert)                          │
│ SENSITIVE DATA:      ⚠️⚠️ Health information (RA 10173) - CONFIDENTIAL        │
│ Note:                Optional - Employee choice to use or not               │
│                      Does not reduce other leave entitlements               │
│ Privacy:             Employer must maintain strict confidentiality           │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 7. EMERGENCY LEAVE (Company Policy)                                         │
├─────────────────────────────────────────────────────────────────────────────┤
│ Default Days:        5 days per year                                        │
│ Carry Forward:       No                                                      │
│ Paid/Unpaid:         UNPAID ✗                                               │
│ For:                 Unexpected personal/family emergencies                 │
│ Approval Level:      Manager                                                │
│ Documentation:       Employer may request explanation/proof                 │
│ Notification:        As soon as possible (immediate preferred)              │
└─────────────────────────────────────────────────────────────────────────────┘
```

## Approval Workflow

### Standard Leave (Vacation, Sick, Emergency)
```
Employee → Manager Approval → HR Processing → Payroll Deduction
```

### Special Leave (Maternity, Paternity, Menstrual Disorder)
```
Employee → HR Documents Review → Medical Verification → HR Approval 
→ Director Notification → Payroll Deduction
```

### Solo Parent Leave
```
Employee → Certificate Verification → HR Approval → Manager 
→ Payroll Deduction
```

## Quick Approval Checklist

### Vacation Leave
- ☐ Employee submitted request in system
- ☐ Balance is sufficient
- ☐ Manager reviewed
- ☐ No scheduling conflicts

### Sick Leave (≤ 2 days)
- ☐ Employee submitted request
- ☐ Balance is sufficient
- ☐ Manager approved

### Sick Leave (> 2 days)
- ☐ Employee submitted request
- ☐ Medical certificate provided
- ☐ HR reviewed documentation
- ☐ Balance is sufficient
- ☐ Manager approved

### Maternity Leave
- ☐ Birth certificate/medical documents submitted
- ☐ Employee is female
- ☐ Notification given 2 weeks in advance (if possible)
- ☐ HR verified leave dates (within 5 years of birth)
- ☐ Director approved
- ☐ Payroll notified

### Paternity Leave
- ☐ Birth/adoption certificate submitted
- ☐ Claimed within 60 days of childbirth/adoption
- ☐ Employee is male
- ☐ HR verified documentation
- ☐ Director approved
- ☐ Payroll notified

### Solo Parent Leave
- ☐ DSWD Certificate or LGU registration provided
- ☐ Solo parent status verified by HR
- ☐ Balance is sufficient
- ☐ Manager approved

### Menstrual Disorder Leave
- ☐ Medical certificate from licensed physician provided (one-time)
- ☐ Diagnosis of menstrual disorder documented
- ☐ HR verified (kept confidential)
- ☐ Manager approved
- ☐ Medical file locked (restricted access only)

## Key Calculations

### Total Yearly Leave Entitlement (Minimum)

**For Regular Employee (Non-Solo Parent, No Health Conditions):**
- Vacation: 15 days
- Sick: 15 days
- Emergency: 5 days (unpaid)
- **Total Paid: 30 days/year**

**For Female Employee Eligible for All Benefits:**
- Vacation: 15 days
- Sick: 15 days
- Menstrual Disorder: 3 days
- Emergency: 5 days (unpaid)
- **Total Paid: 33 days/year**

**For Solo Parent (Non-Maternity):**
- Vacation: 15 days
- Sick: 15 days
- Solo Parent: 5 days
- Emergency: 5 days (unpaid)
- **Total Paid: 35 days/year**

**For Female Solo Parent (Maternity Eligible):**
- Vacation: 15 days
- Sick: 15 days
- Solo Parent: 5 days
- Maternity: 120 days (when applicable)
- Menstrual Disorder: 3 days
- Emergency: 5 days (unpaid)
- **Total Paid: 163+ days/year (when maternity taken)**

**For Male Solo Parent (Paternity Eligible):**
- Vacation: 15 days
- Sick: 15 days
- Solo Parent: 5 days (already at 35)
- Paternity: 14 days (14 days for solo parents, cumulative with solo parent leave)
- Emergency: 5 days (unpaid)
- **Total Paid: 49 days/year (when paternity taken)**

## Data Privacy Reminders (RA 10173)

### SENSITIVE LEAVE TYPES
⚠️ The following contain or reveal sensitive personal information:

1. **Sick Leave** - Reveals health conditions
2. **Maternity Leave** - Reveals pregnancy/reproductive status
3. **Paternity Leave** - Reveals having children/family status
4. **Solo Parent Leave** - Reveals family composition/marital status
5. **Menstrual Disorder Leave** - Reveals reproductive health issues

### Access Control Required
- ❌ Cannot be discussed publicly
- ❌ Cannot be viewed by non-HR staff
- ❌ Limited employee visibility (own leave only)
- ❌ Medical documents locked in HR safe/secure system
- ✓ Audit logs required for all access
- ✓ Data access restricted to authorized HR personnel only

### Document Handling
- Medical documents: Scanned and saved in encrypted HR files
- Original documents: Kept in locked HR file
- Access: HR Manager + HR Director only
- Disposal: Shred after 3 years per company policy
- Requests: Subject to RTI (Right to Information) with privacy redactions

## Common Issues & Solutions

| Issue | Resolution |
|-------|-----------|
| Employee asking for maternity at 6 years postpartum | Explain 5-year limit - must be claimed within 5 years |
| Paternity claim at 70 days after birth | Advise 60-day deadline law - cannot approve |
| Solo parent without DSWD cert | Cannot approve until DSWD registration obtained |
| Menstrual disorder leave without medical cert | Cannot approve until physician certification provided |
| Employee requesting both paternity + solo parent | Both are approved (7+5=12 base, 14+5=19 for solo parents) |
| Carry-forward of sick leave >5 days | Only 5 days can carry forward; excess is forfeited |
| Negative leave balance | Only allow if special company policy exists |
